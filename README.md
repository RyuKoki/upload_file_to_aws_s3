# ☁️ S3 File Manager

แอปพลิเคชันเว็บแบบหน้าเดียว (Single-Page Application) สำหรับจัดการไฟล์บน **Amazon S3** ที่ใช้งานง่าย สวยงาม และรองรับภาษาไทย พัฒนาด้วยเทคโนโลยีเว็บมาตรฐาน (HTML/JavaScript) และ AWS SDK

## ✨ คุณสมบัติ
- **เชื่อมต่อ AWS S3**: รองรับการกำหนด Access Key, Secret Key, Region และ Bucket Name
- **อัปโหลดไฟล์**: รองรับการลากและวาง (Drag & Drop) หรือคลิกเลือกไฟล์
- **จัดการไฟล์**: แสดงรายการไฟล์พร้อมขนาดและวันที่แก้ไขล่าสุด
- **เข้าถึงไฟล์**: สร้าง Signed URL สำหรับเปิดดูหรือดาวน์โหลดไฟล์ได้อย่างปลอดภัย
- **แชร์ไฟล์**: คัดลอก Signed URL (หมดอายุใน 1 ชั่วโมง) เพื่อแชร์ให้ผู้อื่น

## 🚀 วิธีเริ่มต้นใช้งาน
1. เปิดไฟล์ `s3-file-manager.html` ด้วยเว็บเบราว์เซอร์ (เช่น Chrome, Edge, Firefox)
2. กรอกข้อมูล **AWS Credentials**:
   - `Access Key ID`
   - `Secret Access Key`
   - `Region` (เช่น `ap-southeast-1` สำหรับสิงคโปร์ หรือ `ap-southeast-7` สำหรับไทย)
   - `Bucket Name`
3. คลิกปุ่ม **"⚡ เชื่อมต่อ S3"**
4. เมื่อเชื่อมต่อสำเร็จ คุณสามารถเริ่มอัปโหลดไฟล์หรือจัดการไฟล์ที่มีอยู่ได้ทันที

## ⚠️ คำแนะนำด้านความปลอดภัย
- **ห้ามแชร์ข้อมูล Access Key และ Secret Key**: ข้อมูลเหล่านี้มีความสำคัญมาก หากรั่วไหลผู้อื่นอาจเข้าถึงทรัพยากร AWS ของคุณได้
- **การลบไฟล์**: การลบไฟล์ผ่านแอปพลิเคชันนี้จะเป็นการลบถาวร กรุณาตรวจสอบให้แน่ใจก่อนดำเนินการ
- **Signed URL**: URL ที่แชร์ออกมาจะสามารถเข้าถึงไฟล์ได้โดยไม่ต้องใช้รหัสผ่านจนกว่าจะหมดอายุ
